Every blog needs a first post, so here's mine.

I built this with [Jekyll](https://jekyllrb.com/) and deployed it on [Cloudflare Pages](https://pages.cloudflare.com/). The goal was something I could actually maintain — no CMS, no database, just markdown files I can edit anywhere.

The setup is simple: write a post, push to GitHub, Cloudflare handles the rest. That's it.

I'll be using this to write about projects I'm working on. More soon.
